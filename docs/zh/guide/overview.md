# 概览

Z3r0 是一个开源的红队协作工作台，围绕多专家智能体协同构建，面向授权渗透测试、漏洞挖掘、代码审计和安全研究等专业场景。

平台借鉴真实红队的分工模型，由主控智能体协调情报侦察、渗透测试、代码审计、逆向分析和密码分析等专家智能体。任务执行过程中，系统会持续沉淀资产、关系、漏洞发现和攻击路径，形成可长期留存的结构化证据，使安全工作具备更好的可观察性、可审计性和可复现性。

## 核心能力

| 能力 | 说明 |
| --- | --- |
| 多智能体红队编排 | 由主控智能体协调专家智能体，将情报、漏洞、分析和路径梳理拆解为可并行推进的任务。 |
| 会话级运行时架构 | 每个会话绑定独立的智能体编排图和工具快照，支持中断、取消、恢复与连续执行。 |
| 后台子智能体任务 | 子智能体可作为后台任务持久运行，并在完成后唤醒父智能体进行结果整合和后续规划。 |
| 异步沙箱任务系统 | 长时间运行的命令以异步任务执行并持久化状态，避免工具执行阻塞主流程。 |
| 受控沙箱执行环境 | 技能加载、命令执行和输出读取均封装在沙箱工具链内，兼顾隔离执行和结果追踪。 |
| 分布式测试管理器 | 支持多主机、多镜像和多容器管理，为并行测试、环境隔离和资源调度提供基础设施。 |
| 代理出口环境隔离 | 沙箱容器可绑定 HTTP、HTTPS 和 SOCKS5 代理出口，降低操作者环境暴露风险。 |
| 项目化红队工作流 | 通过 WorkProject 统一管理资产、漏洞发现、关系图谱和攻击路径，使过程更易追踪和审查。 |
| 可回放事件时间线 | 持续记录对话、工具调用、子任务、错误和结果事件，支持实时展示与历史回放。 |

## 项目架构

```mermaid
flowchart TB
  Operator["授权安全人员"]
  API["FastAPI 控制平面"]
  Runtime["智能体会话运行时"]
  Graph["会话智能体编排图"]
  Agents["主控与专家智能体"]
  Tools["工具编排层"]
  Sandbox["分布式沙箱资源"]
  Hosts["多主机 / 多容器 / 代理出口"]
  Project["工作项目记录"]
  Evidence["资产 / 漏洞发现 / 攻击路径"]
  Async["异步任务通知"]
  Timeline["事件时间线"]
  Store[("PostgreSQL 持久化")]

  Operator --> API --> Runtime --> Graph --> Agents --> Tools

  Tools --> Sandbox --> Hosts --> Store
  Tools --> Project --> Evidence --> Store
  Tools --> Async --> Store
  Runtime --> Timeline --> Store
```

该架构以 FastAPI 作为控制平面，统一管理会话、项目和执行资源。核心运行时由智能体会话驱动，通过会话智能体编排图组织主控智能体与专家智能体，并由工具编排层连接沙箱执行、项目记录、异步任务和事件时间线。分布式沙箱资源为授权安全测试提供隔离执行环境，WorkProject 则将资产、漏洞发现和攻击路径沉淀为可追踪、可复盘的项目证据。底层通过 PostgreSQL 持久化会话状态、任务进度、漏洞发现、资产关系、攻击路径和回放事件，确保整体流程可追踪、可审查。

## 运行时序

```mermaid
sequenceDiagram
  participant UI as 前端工作台
  participant API as 会话 REST 接口
  participant WS as 会话事件流
  participant Pool as 会话池
  participant Sess as 智能体会话驱动
  participant SDK as Runner + 会话记忆
  participant Tool as 挂载工具
  participant DB as PostgreSQL

  UI->>API: 提交 turn / 中断 / 取消
  API->>Pool: 提交会话输入
  Pool->>Sess: 启动或恢复 turn
  Sess->>DB: 标记运行 + 加载事件历史
  UI->>WS: 订阅会话事件
  Sess->>SDK: 流式运行智能体

  SDK->>Tool: 调用工具

  alt 项目记录 / 知识库
    Tool->>DB: 读写结构化状态
    DB-->>Tool: 返回结果
  else 沙箱同步命令
    Tool->>Tool: 在已选沙箱执行命令
    Tool-->>SDK: 返回输出元数据
  else 沙箱异步命令 / 子智能体
    Tool->>DB: 创建任务 + 等待通知
    Tool-->>SDK: 返回运行中
    DB-->>Pool: 完成后通知变为待处理
    Pool->>Sess: 自动恢复会话
  end

  SDK-->>Sess: 流式事件
  Sess->>DB: 持久化时间线
  Sess-->>WS: 标准化事件
  WS-->>UI: 实时展示
```
