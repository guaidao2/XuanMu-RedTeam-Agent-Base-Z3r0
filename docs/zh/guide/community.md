# 组建社群

Z3r0 取自 “Zero” 的变体写法。在安全研究和技术工作中，Zero 既指向从零开始的探索，也指向未知漏洞、未知边界和未知系统行为的发现过程。

借着这个项目，我想要组建一个小而精的社群，来连接一批真正专注于技术、始终保持好奇心的人。

## 社群理念

这里更关注深入的问题讨论和严谨的验证实践，而不是浅层的讨论和工具的堆砌。

**十分欢迎以下几类人加入：**

1. 对网络安全、漏洞研究、代码审计、红队工程等领域感兴趣的开发者或研究者；
2. CTF 选手、SRC 贡献者或对智能体应用、人工智能技术充满兴趣的开发者；
3. 热爱工程技术、致力于前沿技术探索的安全行业或互联网行业从业者；

**以下几类人请勿打扰：**

1. 黑/灰产从业者、键政玩家；
2. 戾气极重、素质低下、肆意发泄情绪的人；
3. 非法使用工具发起恶意攻击的人；

Z3r0 正快速迭代中，诚挚邀请志同道合的开发者加入到项目的开发工作中，共同探讨技术话题。
